FSContext.Reload
================
The Reload function is used to discard and load the same context.

------
The function definition can be found in the following file:

    Model/CardSet/MContext.swift


Syntax
------
.. code-block:: c

    Reload() throws -> Void


Parameters
----------
None.

Return Value
------------
None.

Remarks
-------
* The old set entry list will be discarded even if the context load failed.
