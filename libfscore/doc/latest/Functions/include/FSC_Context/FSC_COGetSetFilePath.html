FSC_COGetSetFilePath¶

The FSC_COGetSetFilePath function returns the file path for a set entry contained in a given context.

Header¶

The function definition can be found in the following file:

include/FSC_Context.h

Syntax¶

const char* FSC_NULLABLE FSC_COGetSetFilePath(FSC_ContextOpaque* FSC_NONNULL p_Context,
                                              size_t us_Set);

Parameters¶

Parameter

Description

p_Context

The context to check.

us_Set

The set to get the file path for.

Return Value¶

The UTF-8 file path string on success, NULL on failure.

Remarks¶

  • The returned string is always null terminated.

Code Examples¶

None.
