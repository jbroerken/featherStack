FSC_CD_STR_MAX_LENGTH
=====================
The FSC_CD_STR_MAX_LENGTH macro is used to define the maximum string length for
strings recieved as parameters.

Header
------
The macro definition can be found in the following file:

.. code-block:: c

    src/FSC_CardData.h


Definition
----------
The defined macro is as follows:

.. code-block:: c

    #define FSC_CD_STR_MAX_LENGTH 511
