FSC_CSD_STR_BUFFER_LENGTH
=========================
The FSC_CSD_STR_BUFFER_LENGTH macro is used to define the maximum buffer length 
in bytes for strings.

Header
------
The macro definition can be found in the following file:

.. code-block:: c

    src/FSC_CardSetData.h


Definition
----------
The defined macro is as follows:

.. code-block:: c

    #define FSC_CSD_STR_BUFFER_LENGTH 512
